# Cloud.md - Dokumentasi Desain Website Kenangan Hidup

## Deskripsi Website
Website ini adalah sebuah platform pribadi untuk menyimpan dan menampilkan seluruh kenangan hidup pemilik website, mulai dari masa kecil, remaja, hingga kehidupan saat ini dan masa depan. Website dibuat dengan konsep sederhana, elegan, hangat, dan penuh nostalgia.

Tujuan utama website ini bukan sebagai media sosial, melainkan sebagai tempat arsip kehidupan pribadi berbentuk digital yang bisa diakses secara publik dalam mode baca.

---

# Konsep Utama Website

Website harus memberikan kesan:

- Elegan
- Bersih
- Nostalgia
- Personal
- Hangat
- Premium sederhana

Website tidak boleh terlihat terlalu kosong, namun juga tidak boleh terlalu padat.

Semua elemen harus memiliki keseimbangan jarak yang pas.

---

# Role User

## 1. Free User (Tanpa Login)
User biasa hanya dapat:

- Melihat kenangan
- Membuka detail cerita
- Melihat foto/video
- Membaca timeline kehidupan
- Melihat galeri
- Mencari kenangan tertentu
- Melihat kategori kenangan

User tidak memiliki akses upload/edit/delete.

---

## 2. Admin (Pemilik Website)
Admin adalah pemilik website dan memiliki akses penuh.

Fitur admin:

- Login admin
- Upload kenangan baru
- Edit kenangan
- Hapus kenangan
- Upload foto
- Upload video
- Membuat kategori
- Mengatur timeline kehidupan
- Menandai kenangan favorit
- Mengatur tampilan homepage
- Mengatur urutan kenangan
